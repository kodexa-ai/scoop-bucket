# Kodexa Scoop Bucket

**Copyright © 2026 Kodexa, Inc. All rights reserved.**

Official Scoop bucket for Kodexa CLI tools (Windows).

## Installation

```powershell
# Add the bucket
scoop bucket add kodexa https://github.com/kodexa-ai/scoop-bucket

# Install kdx CLI
scoop install kdx

# Update kdx
scoop update kdx
```

## Available Apps

- **kdx** - Kodexa CLI for managing Kodexa AI Platform resources

## About

This bucket provides easy installation of Kodexa command-line tools on Windows systems using [Scoop](https://scoop.sh/).

### What is Scoop?

Scoop is a command-line installer for Windows. If you don't have Scoop installed:

```powershell
# Install Scoop
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
irm get.scoop.sh | iex
```

For more information, visit [kodexa.ai](https://kodexa.ai).
